# Shrividya Hegde — Career Profile Site

Personal portfolio site built with Jekyll, hosted on GitHub Pages at **[shrividya.github.io](https://shrividya.github.io)**.

The site has two views — switch between them using the toggle bar at the top of the page.

---

## ⚡ Airflow DAG Mode

Career presented as a live Apache Airflow data pipeline. Each stage of the pipeline maps to a phase of my career:

| Task Node | What it covers |
|---|---|
| 📥 `ingest_sdet_roots` | SDET years — TCS, KPMG, CBA (2014–2022) |
| ⚙️ `transform_astronomer` | Astronomer + Apache Airflow OSS (2022–present) |
| ✅ `validate_certs_oss` | Certifications, education, and recognition |
| 🧠 `ai_rag_projects` | AI/RAG projects and conference talks |
| 🚀 `serve_community` | Community, writing, and connect |

**How to navigate:**
- Click any **task node** in the graph row, or a **DAG run** in the left sidebar, to open that stage
- Each stage shows a timestamped **log output**, impact stats, and a task instance table
- Green `● success` = completed career phase · Blue `● running` = ongoing

## 👤 Normal Profile Mode

A clean, familiar profile layout for visitors who aren't data engineers:

- Bio, location, and social links at the top
- Live GitHub stats — contribution streak, top languages, trophies
- Full experience timeline with impact metrics
- Certifications, featured projects with GitHub links
- Technical writing, community work, and connect cards
